## 📥 نصب و راه‌اندازی

# دریافت اسکریپت
```
bash <(curl -Ls https://raw.githubusercontent.com/hodacloud/AbuseTerminator/refs/heads/main/AbuseTerminator.sh)
```

# قابل اجرا کردن
```
chmod +x AbuTerminator.sh
```
# اجرا با دسترسی روت
```
sudo ./AbuTerminator.sh
```
